之前裝 Decky Loader 設定了 deck 使用者的密碼，結果現在要用 sudo 的時候發現忘記了。參考 Reddit 的步驟成功重置了。步驟如下：

  1. 將 Steam Deck 關機
  2. 接上鍵盤
  3. 按住主機右下三個點的按鈕後，按一下電源鍵
  4. 成功的話應該會進入 Recovery Menu
  5. 選擇 Current SteamOS + Boot Menu（應該是第三選項）
  6. 接者會出現 GRUB Menu，停在 SteamOS 選項上（應該是第一選項）按下 E 執行 [E] Edit Selection
  7. 會出現許多命令，用方向鍵往下捲，直到停在 steamenv_boot 開頭的一行上
  8. 按下 End 鍵到 steamenv_boot 命令的最後，加上 - systemd.debug_shell
  9. 按下 Ctrl+X 重開機
  10. 到主畫面按下 Ctrl+Alt+F9 開啟 Debug Shell
  11. 輸入 passwd deck 後輸入新密碼兩次
  12. 成功後 Ctrl+Alt+F1 關閉 Debug Shell

這樣應該就可以了。
