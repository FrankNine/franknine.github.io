昨天剛剛把站從 Media Temple 搬到 Linode ．

主要的原因是 Media Temple 每個月 $20 硬是 Linode 的兩倍，現在沒有在賺美金了要省著點用．

另一方面是 Linode 我拿到 Tokyo 的節點，在台灣讀取起來的速度比 Media Temple 好很多．

不過 Linode 就沒有像 Media Temple 那樣幫你準備好好的．

Linode 刷下去後又花了一個多禮拜在架 LERP 跟 WordPress ．

然後 WordPress Importer 又把圖跟 Plugin 弄掉了…

我真的是脫節寫網站太久了，沒想到現在 Apache 已經開始失寵了．

新手上路，希望 server 有設定錯能麻煩提點一下，不要爆我菊．拜託拜託．

網站還留著，也就是說，嗯，我要再寫 blog 了．

停刊超過一年真的是富樫級的了．

如果還有人在看，我想說聲：

你好，謝謝你．
