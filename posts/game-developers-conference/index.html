現在是美國西岸的 3/18 星期五的晚上，Game Developer Conference 16 應該是正式的告一段落了。自從回來台灣之後就再也沒回過美國了，自然最近幾屆 GDC 也都沒有參加。今天下午睡覺又被樓上鄰居吵起來之後，想趁這個時間寫寫我自己關於 GDC 的雜談。

Game Developer Conference 從 1988 年到現在已經有 27 年的歷史，現在由英國 UBM 集團舉辦，是目前全球最大的遊戲開發者面向峰會。主會通常是辦在每年三月，為期一週五天。另外還有 GDC Europe 跟 GDC China ，但是我只有參加過在美國的 GDC 本會，所以以下 GDC 所指的都是在美國舊金山舉辦的那個。另外值得一提的是 UBM 集團同時營運可以算是全球最大的遊戲開發者交流網站 Gamasutra （Sean 兄很好心經營的英文遊戲資料翻譯網站遊戲思維很多原文就是來自 Gamasutra），然後還有 Game Developer Magazine （現已停刊，但是 UMB 很慷慨的把從 1994 到現在所有過去雜誌都掃描上傳在這裡）

我自己參加的第一屆 GDC 是在 2011 年，當時必須要翹課去參加，白天逛大街晚上在青年旅館的餐廳裡趕作業（睡通鋪沒有桌子）。雖然說有夠慘的，但是絲毫無損於第一次參加所受到的震撼與衝擊。11 年我買的是 Main Conference Pass ，從第三天才能入場，一開始就是去年過世的任天堂前社長岩田聰的 Keynote ，然後是 World of Warcraft 的設計師 Tom Chilton 談 WOW 4.0 的設計理念，晚上是 Minecraft 的 Notch 獲頒 Independent Games Festival 跟 Game Developers Choice Awards 各項大獎。隔天是上帝也抓狂的作者 Peter Molyneux 分享他從賣罐裝烤豆起家到不小心變成 God Game 之父的曲折故事。再隔天是洞窟物語的作者天谷大輔先生演講開發的心路歷程，還分享了從來沒有曝光過的 prototype 影片。個人當了兩年的 WOW 廢人，玩過一堆老任的遊戲，然後又是 Minecraft 、上帝也抓狂跟洞窟物語的腦殘粉，應該不難想像第一次參加時的我是有多嗨了。有機會能見到你心目中所有想得到的遊戲界人物，這是 Game Developer Conference 最顯著吸引力。

解釋一下 Game Developer Conference 的票價跟門票種類。

GDC 舉辦的五天前兩天是 Summit、 Tutorial 跟 Bootcamp，後三天是 Main Conference 。前兩天的 Summit 是圍繞在特定主題小 Conference ，主題有 AI 、營運、 eSport 、手機遊戲還有獨立遊戲等等。不過 Summit 主題每年都會變，像是今年有新的 VR Summit 。持有 Summit Pass 的人可以在前兩天在各個 Summit 之間切換，對於獨立開發者 GDC 有特別提供只能參加 Indie Summit 的 Indie Pass ，不過這種票非常搶手，如果有興趣的人要盯著 GDC 售票網站搶票。Tutorial 跟 Bootcamp 通常是預設你這兩天會聽到底的教學課程，我自己是沒有參加過，不過好像有台灣朋友有參加過 Game Design Workshop ，不知道有沒有心得分享。

後三天是 Main Conference ，基本上程式、美術、設計、音樂什麼主題都有。參加者可以隨自己喜好切換，其實有時候也不太知道某個演講是歸類在 Summit 還是 Main Conference 的標準在哪裡。有遇過同一個講者講同一個遊戲結果兩個演講一個在 Summit 一個在 Main Conference 。這時候你就會感受感受到運營的惡意了～

GDC 的門票高得嚇人，我覺得他們的定位應該是假設大部分人會跟公司報帳。如果想要一些折扣的話可以買早鳥票，不過早鳥票在賣的時候通常議程都還沒有確定，所以就是有點在賭的感覺。然後你去年參加過，今年可以在早鳥時間買更優惠的 Alumni Pass 。加入 IGDA （International Game Developer Association）也能有一定的折扣，如果你不在意去 GDC 要幫忙顧場地的話可以申請 Conference Associate ，如果申請過基本上就是要在展出時間做滿一定時間的志工，剩下的時間 Conference Associate 是視同最高級的 All Access Pass ，不過我沒有申請上過，所以沒有這方面的經驗可以分享。

同時從週三開始後三天展場也會開放，除了 Student Pass 只能最後一天參加之外其他所有 Pass 都能逛三天。展場主要分做 Expo 跟 Career Center 。Expo 就是給開發者看的電玩展，會有各家遊戲硬體、引擎廠商參展。可以去搶最新硬體的嚐鮮，或是搶會場限定紀念品。

Career Center 就是求職場所，不過我自己去 GDC 找工作的經驗就是所有公司都是在找挖角，對於應屆畢業生的興趣沒有這麼大。我自己是沒有在 GDC  成功找到任何工作機會或是實習機會，如果對於成功案例有興趣的，可以參考 CJCat 的心得分享。剛好今天他把英文版的求職經歷翻成中文分享出來。

然後如果你現在在美國念書想進遊戲業，想在 GDC Career Center 找工作，絕對不要買 Student Pass。Student Pass 看起來非常便宜，但是它只有最後一天能進展場。通常到了最後一天展場人員都累了，沒有什麼力氣跟你深談，甚至有些公司最後一天會直接撤展。如果你要找工作的話還是買高級一點的 Pass ，然後越早去越好。

展場除了 Expo 跟 Career Center 這兩個主要展覽之外還有像是 IGF Pavilion 跟 Alt.Ctrl.GDC 這種特別的區域。IGF Pavilion 基本上就是當年 IGF 入圍的參賽者的展示空間，如果跟獨立遊戲熟的話，當然不能錯過跟開發者攀談的機會。

Alt.Ctrl.GDC 是 2014 才新增的區域，主要是展覽新穎的遊戲控制裝置，所謂的新穎不是接個 wii remote 或是 Kinect 就算了，而是：

用發票機製作的文字冒險遊戲 https://choosatron.com/

這個有十六鍵的整人對戰遊戲作者是塊魂的作者高橋慶太，遊戲官網在這裡。

感覺表面上的 GDC 介紹就花了不少時間寫，現在就先斷在這裡，之後再寫比較個人一點的心得吧。
