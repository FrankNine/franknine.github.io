There are JavaScript debuggers for each browser out there. A debugger can be invaluable for examining variables, or seeing what parameters are available in a class and what their class types are. The Chrome debugger is described here, Firefox’s Firebug debugger here, Safari here. Also see this great post for more help.

For more serious debugging of the online exercises, see the invaluable advice here from Mathieu Civel. A small addition to your code lets you set a break and examine values easily within your browser.
